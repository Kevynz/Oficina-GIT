Introdução ao JavaScript

O JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web. Ela permite que você crie interatividade, manipule elementos da página e responda a eventos do usuário.

1. O que é JavaScript?

O JavaScript é uma linguagem de script que roda no navegador do usuário. Ele permite que você adicione comportamentos dinâmicos às páginas da web. Por exemplo, você pode validar formulários, criar animações e etc.

2. Objetos em JavaScript

Em JavaScript, objetos são estruturas de dados que representam algo. Eles têm propriedades (chave-valor) e métodos (funções). Um exemplo comum é o objeto document, que representa a página HTML atual. Você pode acessar elementos da página usando document.getElementById('elementId').

3. Decisões (Condicionais)

As estruturas de decisão permitem que seu código tome diferentes caminhos com base em condições. Os principais tipos de condicionais em JavaScript são:

  • if: Executa um bloco de código se a condição for verdadeira.
  • else: Executa um bloco de código alternativo se a condição do if não for verdadeira.
  • else if: Permite testar várias condições em sequência.
  • switch: Uma forma mais organizada de lidar com múltiplas opções.

4. Laços de Repetição

Os laços de repetição permitem que você execute um bloco de código várias vezes. Os principais são:

  • for: Executa um bloco de código um número específico de vezes.
  • while: Executa um bloco de código enquanto uma condição for verdadeira.
  • do...while: Semelhante ao while, mas garante que o bloco seja executado pelo menos uma vez.

5. Variáveis

Em JavaScript, as variáveis são usadas para armazenar valores de dados. O JavaScript oferece suporte a várias maneiras de declarar variáveis, incluindo o uso de:

  • Var - Possui escopo de função, o que significa que a variável declarada com var é acessível em toda a função em que foi declarada, independentemente do bloco em que está contida.
  • Let - Permite declarar variáveis ​​com escopo de bloco. Isso significa que as variáveis declaradas com let são acessíveis apenas dentro do bloco em que são definidas.
  • Const - Usada para declarar constantes em JavaScript. O valor de uma constante não pode ser alterado após a sua atribuição inicial, e ela também possui escopo de bloco.

Var é mais antigo, com escopo mais amplo e comportamento de hoisting, o que pode causar confusão. Em muitos casos, let e const são preferidos por oferecerem escopos mais previsíveis.
